# Ritchie Formula

## References

This formula as been implemented base on [https://github.com/Malfrats/xeuledoc](https://github.com/Malfrats/xeuledoc)

## Command

```bash
rit googledoc get insights
```

## Description

A formula to fetch information about any public Google document.
It's working on :

- Google Docs
- Google Spreadsheets
- Google Slides
- Google Drawning
- Google My Maps
- Google Apps Script
- Google Jamboard
